## Implementation notes

### Grouping order (top to bottom)
in_progress → booked → quoted → awaiting_payment → no_show → **paid (collapsed)** → **cancelled (collapsed)** → **written_off (collapsed)**

Only render groups that have at least 1 job. If a group is empty, skip it entirely.

### Status dot colours
```
in_progress: #15803D
booked:      #1D4ED8
quoted:      #7C3AED
awaiting:    #B45309
no_show:     #92400E
paid:        #9CA3AF
cancelled:   #D1D5DB
written_off: #D1D5DB
```

### Terminal group collapse
- paid, cancelled, written_off are collapsed by default
- Tap the collapsed row → expand to show job rows
- Use local state: `const [expanded, setExpanded] = useState<Set<string>>(new Set())`

### Filter chips
- **All**: all groups in order above
- **Active**: in_progress + booked groups only
- **Unpaid**: awaiting_payment only, sorted: overdue (invoice_sent_at ≥ 30d) first, then by invoice_sent_at ascending

### Contextual sub-line per status
```
in_progress:       "{date} · {elapsed time}in"  e.g. "Wed 14 May · 2h 14m in"
booked:            "{date} · {time}"             e.g. "Wed 14 May · 2pm"
quoted:            "Sent {n} days ago" or "Sent today"
awaiting_payment:  "Invoice sent {n} days ago" (+ flag badge inline)
no_show:           "{date} · {time}"
paid:              "Paid {date}"
cancelled:         "Cancelled {date}"
written_off:       "Written off {date}"
```

### Flag badge inline (in sub-line)
Show flag badge inside `.job-sub` next to the text:
- awaiting_payment + ≥30d: Overdue·Xd (red)
- awaiting_payment + 1-29d: Chase·Xd (amber)
- awaiting_payment + 0d: no flag
- no_show: "Action needed" (amber)
- enquiry + <2h: "New" (blue)

Use the `FlagBadge` component from `src/components/FlagBadge`.

### Dexie query
```typescript
// Load all jobs with their customers and totals
const jobs = await db.jobs.where('user_id').equals(userId).toArray();
const customers = await db.customers.where('user_id').equals(userId).toArray();
const lineItems = await db.line_items.toArray();

// Build customer map and total map
const customerMap = Object.fromEntries(customers.map(c => [c.id, c]));
const totalMap: Record<string, number> = {};
for (const item of lineItems) {
  totalMap[item.job_id] = (totalMap[item.job_id] || 0) + item.amount;
}
```

### Row tap
```typescript
navigate(`/jobs/${job.id}`)
```

Two equal-width buttons (same as Home screen):
- "+ New Quote" (primary style) → `navigate('/quote')`
- "Log Missed Call" → `navigate('/quote', { state: { entryPoint: 'missed_call' } })`
