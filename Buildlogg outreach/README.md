## Daily Metrics

Auto-generated by `outreach-metrics.ts` (runs via launchd at 14:00 BST). Reports saved to `logs/outreach-metrics-YYYY-MM-DD.md`. Each report includes:
- Funnel summary (sent → delivered → opened → clicked → signed up)
- Per-trade performance
- Domain deliverability
- Send time analysis
- Lead score vs engagement
- PostHog sign-up events
- Suppression/bounce tracking

## Design Iterations

See `design/email-design-test-log.md` for the full history of 11 design versions, what broke, what fixed it, and A/B subject line results.

Key findings:
1. **Font quotes** — `font-family:"Manrope"` breaks `style="..."` attribute → ALL CSS lost. Fix: single quotes.
2. **Apple Mail buttons** — need table-based layout, not `inline-block` + `line-height`.
3. **Dark footer** — stole attention from content. Replaced with light footer matching confirmation email.
4. **Cold emails force light mode** — `color-scheme: light only` meta prevents dark mode inversion.

## Sending Commands

```bash
# Preview eligible leads (no send)
cd ~/lead-triage && npx tsx src/scripts/send-cold-emails.ts preview ~/lead-triage/data/beauty_leads_export.csv 1 --vertical=beauty

# Send beauty batch (step 1)
cd ~/lead-triage && npx tsx src/scripts/send-cold-emails.ts send ~/lead-triage/data/beauty_leads_export.csv 1 --vertical=beauty

# Send trades batch (step 1)
cd ~/lead-triage && npx tsx src/scripts/send-cold-emails.ts send 1

# Send follow-up (step 2 — Day 3)
cd ~/lead-triage && npx tsx src/scripts/send-cold-emails.ts send 2 --vertical=beauty

# Campaign stats
cd ~/lead-triage && npx tsx src/scripts/send-cold-emails.ts stats

# Daily metrics report
cd ~/lead-triage && npx tsx src/scripts/outreach-metrics.ts
```
